3.	Управление техникой и машинами на стройке
Описание: Компания занимается строительством зданий. У каждого объекта строительства есть этажность и площадь. Каждый объект имеет фазы строительства, требующие различных машин и механизмов на м2 строящейся площади (бульдозеры, самосвалы, краны, подъемники, сварочные аппараты, штукатурные станции).
Функциональность:
●	Создание объектов строительства и управление их состояниями.
●	Закупка техники.
●	Направление техники на объекты.
●	Моделирование процесса строительства пошаговое, один шаг – неделя.
●	Учесть, что у компании может быть несколько объектов одновременно
●	Строительство может быть приостановлено
●	Подсчет (статистика) затраченных денег и материалов на строительство
